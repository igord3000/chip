# Chip Agent

Минимальный AI coding-агент с GUI, кэшированием и расширяемой архитектурой.

## Быстрый старт

```bash
# Установить
git clone https://github.com/igord3000/chip.git
cd chip
pip install -e .

# Настроить (скачает Ollama и модель)
chip -s

# Запустить
chip
```

## Команды

| Команда | Описание |
|---------|----------|
| `chip` | Запуск GUI (по умолчанию) |
| `chip -c` | CLI чат |
| `chip -c "задача"` | CLI с задачей |
| `chip -m MODEL` | Указать модель |
| `chip -r` | Продолжить сессию |
| `chip -s` | Настройка |
| `chip --status` | Статус системы |

## Комбинации флагов

```bash
chip                          # GUI
chip -c                       # CLI чат
chip -c "Hello"               # CLI + задача
chip -c -m qwen3:8b           # CLI + модель
chip -c -m qwen3:8b "код"     # CLI + модель + задача
chip -c --resume              # CLI + продолжить
chip -c -r -m qwen3:14b       # Всё вместе
```

## Возможности

- **GUI** — современный интерфейс в терминале (Textual)
- **CLI** — классический чат в терминале
- **Кэширование** — экономия токенов
- **Память** — долгосрочная память между сессиями
- **Инструменты** — bash, файлы, интернет, субагенты
- **Контекст** — отслеживание токенов

## Инструменты

| Инструмент | Описание |
|------------|----------|
| `bash` | Выполнение команд |
| `read_file` | Чтение файлов |
| `write_file` | Запись файлов |
| `list_files` | Список файлов |
| `web_search` | Поиск в интернете |
| `web_fetch` | Чтение страниц |
| `download` | Скачивание файлов |
| `subagent` | Параллельные задачи |

## Конфигурация

Файл: `~/.chip/config.json`

```json
{
  "model": "qwen3:8b",
  "max_tokens": 32000,
  "temperature": 0.7
}
```

Или через переменные окружения:

```bash
export LLM_MODEL=qwen3:8b
export LLM_BASE_URL=http://localhost:11434/v1
```

## Горячие клавиши (GUI)

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Q` | Выйти |
| `Ctrl+C` | Очистить чат |
| `Ctrl+S` | Сохранить сессию |
| `Enter` | Отправить сообщение |

## Модули vs Скиллы

**Модули** — Python-код (агент, кэш, инструменты).

**Скиллы** — YAML/JSON конфигурации с инструкциями для агента.

Скиллы будут добавлены позже.
